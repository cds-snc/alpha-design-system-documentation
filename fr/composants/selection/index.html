Sélection
<gcds-select>

Autres noms : menu déroulant, menu de sélection.

Une sélection présente une longue liste d’options parmi lesquelles un seul choix est fait.

Problèmes résolus par la sélection

Utilisez la sélection dans un formulaire afin de :

  • Préciser plus de 7 réponses possibles dans une liste déroulante prédéfinie.

  • Limiter le nombre de réponses admises pour une question.

  • Permettre à une personne de répondre à une question sans devoir rédiger une réponse, en sélectionnant un élément à partir d'une liste.

Composants connexes

Cases à cocher : lorsque vous prévoyez que l'utilisateur·rice sélectionne une ou plusieurs options parmi une liste d'éléments.

Boutons radio : lorsque vous proposez à l'utilisateur·rice un petit ensemble d'options pour une seule sélection connue comportant moins de 7 options.
