Bouton radio

Autres noms : case d’option, groupe radio.

Problèmes résolus par les boutons radio

Utilisez les boutons radio avec un jeu de champs pour effectuer une demande de renseignements à l’aide d’une liste d’options lorsque vous vous attendez à ce que l’utilisateur·rice choisisse une seule réponse pour :

  • Limiter le nombre de réponses possibles à votre question.

  • Déterminer les réponses possibles à votre question.

  • Lui donner la capacité de répondre sans avoir à écrire en sélectionnant une option parmi une liste.

Composants connexes

Cases à cocher lorsque vous prévoyez que l’utilisateur·rice sélectionne plusieurs options parmi une liste d’éléments.

Sélection lorsque vous donnez à l’utilisateur·rice une moyenne ou grande quantité d’options pour une seule option connue.
